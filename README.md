# Policy Site

This folder contains static legal pages for app store compliance.

## Files

- `index.html`
- `privacy-policy.html`
- `data-deletion.html`
- `terms.html`
- `support.html`

## Publish on GitHub Pages

1. Create a public repository (for example `beastaldug-legal`).
2. Copy these files to the repository root.
3. Push to `main`.
4. In repository settings, enable GitHub Pages from `main` branch root.
5. Use this URL in Play Console:
   - `https://<github-username>.github.io/<repo>/privacy-policy.html`

## Important

Replace placeholder emails before publishing:

- `privacy@beastaldug.com`
